Smallest-k Prefix-Sum Decision Algorithm

Analysis of Algorithms

Given an array, A, of n (distinct) numbers, decides whether there exists some positive integer, k, such that you can pick k distinct elements of A whose sum is at most k³. Key idea: if any k works, the k smallest work (minimum possible sum). 

def smallest_k(input_array):
          """
          Decides whether there exists an integer k such that
          the input array contains k numbers whose sum is less than k**3
          """
          if len(input_array) == 0:
              return False

          sorted_array = sorted(input_array)

          running_sum = 0

          for i in range(len(sorted_array)):

              running_sum += sorted_array[i]

              if running_sum <= (i + 1)**3:
                  return True

          return False
